# e-plantShopping

## Paradise Nursery Plant Shop

e-plantShopping is a React and Redux based online plant shopping application developed for Paradise Nursery.

The application allows customers to browse a collection of plants, add products to a shopping cart, update quantities, remove items, and view the total cost of their purchases.

## Features

* Landing page with Paradise Nursery company information
* Attractive background image and company branding
* Product listing page displaying various plants
* Add to Cart functionality
* Shopping Cart page
* Increase and decrease product quantity
* Remove items from cart
* Automatic total cost calculation
* Redux Toolkit state management

## Technologies Used

* React
* Redux Toolkit
* JavaScript (ES6+)
* CSS3
* Vite
